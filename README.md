# vue-collapse-box
This is vue component, which the use for toggle box with slide effect

## Usage
```
<template>
    <collapse-box :open="isActive" :duration="0.75">
        <div class="collapse-block__body">
            Some Content
        </div>
    </collapse-box>
</template>
```

yarn install
